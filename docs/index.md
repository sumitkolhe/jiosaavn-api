## Local development

### Prerequisites

- NodeJS v16+

### Run locally

- Clone the Repo

  ```sh
  git clone https://github.com/sumitkolhe/jiosaavn-api

  cd jiosaavn-api
  ```

- Install the dependencies

  ```sh
  yarn install
  ```

- Start development server

  ```sh
  yarn dev
  ```
